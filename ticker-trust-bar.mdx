---

The **Ticker / Trust Bar** section is an auto-cycling bar that rotates through up to 8 trust signal items — each with an icon and a short text statement. A persistent label sits to the left while the items cycle with dot navigation below. Use it to communicate brand promises such as complimentary shipping, ethical sourcing, lifetime servicing, and free returns.

### How to Add the Section

1. From your Shopify admin, go to **Online Store → Themes** and click **Customize**.
2. Navigate to the page where you want to add this section.
3. Click **Add section** and select **Ticker / Trust bar**.
4. Add **Item** blocks for each trust signal (up to 8 items).
5. Click **Save**.

### Section Settings

#### Content

- **Label:** A persistent left-side label displayed before the cycling items begin (default: "House · Standards").

#### Behaviour

- **Interval:** Time in milliseconds between each automatic item advance (range: 2000–8000ms, step 500ms). Default is 3500ms.

#### Style

- **Color scheme:** Choose a color scheme for the section.

#### Spacing

- **Padding top / Padding bottom:** Control vertical spacing around the section.

---

### Item Block Settings

Each Item block is one trust signal in the ticker. You can add up to 8 items.

#### Icon

- **Image:** Upload a small icon image for this item. Takes priority over the SVG field if both are provided.
- **Icon:** Paste custom SVG path data for the icon (used only if no image is uploaded).

#### Content

- **Text:** The trust signal statement displayed next to the icon (e.g., "Complimentary insured shipping — worldwide").
