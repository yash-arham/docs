The **Search** page in the Sentiment theme is powered by a single **Search** section that renders a full storewide search experience — a breadcrumb bar, an eyebrow label, a large search input, a results grid, and an empty-state prompt. Results per page, column layout, and suggested search terms are all configurable from the Theme Editor.

<Frame>
  ![Search](/images/Search.png)
</Frame>

### How to Configure the Search Page

1. From your Shopify admin, go to **Online Store → Themes** and click **Customize**.
2. From the top page dropdown, select **Search**.
3. Click **Search** in the left sidebar under **Template** to open its settings.
4. Click **Save** when done.

---

### Search Section Settings

<Frame>
  ![Search 1](/images/search-1.png)
</Frame>

#### Layout

- **Enable full width:** Toggle to extend the section edge-to-edge across the viewport.

#### Results

- **Results per page:** Number of search results shown per page before pagination (default: 6). A Load more button reveals the next batch without a full page reload.
- **Columns:** Number of result columns on desktop — **2**, **3**, or **4**.
- **Columns on mobile:** Number of result columns on mobile — **1** or **2**.

#### Suggested Search Terms

- **Terms (comma-separated):** A list of suggested search terms shown below the search field and on the empty-state screen — before the visitor has typed a query (e.g., "rings, gold, bespoke, materials"). Leave blank to hide suggestions.
