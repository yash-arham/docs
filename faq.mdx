The **FAQ** section is a clean, single-column accordion-style list of questions and answers. Each item can optionally include up to two related links within the answer. It's ideal for product pages, policy pages, or any context where a concise list of questions is preferred over a full dedicated FAQ page layout.

### How to Add the Section

1. From your Shopify admin, go to **Online Store → Themes** and click **Customize**.
2. Navigate to the page where you want to add this section.
3. Click **Add section** and select **FAQ**.
4. Add **Item** blocks for each question and answer pair.
5. Click **Save**.

### Section Settings

#### Content

- **Eyebrow:** Small label displayed above the heading.
- **Heading:** Main section heading. Supports rich text and italic emphasis.
- **Heading typography:** Select the typographic style for the heading.
- **Show count:** When enabled, displays the total number of questions in the section.

#### Style

- **Full width:** When enabled, the section stretches edge-to-edge.
- **Color scheme:** Choose a color scheme for the section.

#### Spacing

- **Padding top / Padding bottom:** Control vertical spacing around the section.

---

### Item Block Settings

Each Item block is one accordion question-and-answer pair.

- **Question:** The question text displayed in the accordion header.
- **Answer:** The answer content revealed when the accordion is expanded. Supports rich text formatting.
- **Link 1 label / URL:** An optional first related link displayed within the answer.
- **Link 2 label / URL:** An optional second related link displayed within the answer.
