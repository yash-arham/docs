The **FAQ 2 Column** section presents a FAQ in a two-column layout — the heading and intro text sit in the left column, and the accordion questions expand in the right column. It's a balanced, editorial layout suitable for integrating a FAQ into a broader page without dedicating the full width to it.

</Frame>

### How to Add the Section

1. From your Shopify admin, go to **Online Store → Themes** and click **Customize**.
2. Navigate to the page where you want to add this section.
3. Click **Add section** and select **FAQ - 2 column**.
4. Add **FAQ item** blocks for each question and answer.
5. Click **Save**.

### Section Settings

#### Content

- **Eyebrow:** Small label above the left column heading.
- **Heading:** Main section heading displayed in the left column. Supports rich text and italic emphasis.
- **Heading typography:** Select the typographic style for the heading.
- **Body:** Supporting text displayed below the heading in the left column.
- **CTA label / URL:** An optional link displayed below the accordion in the right column. Leave the URL blank to hide it.

#### Style

- **Full width:** When enabled, the section stretches edge-to-edge.
- **Color scheme:** Choose a color scheme for the section.

#### Spacing

- **Padding top / Padding bottom:** Control vertical spacing around the section.

---

### FAQ Item Block Settings

Each FAQ item block is one accordion question-and-answer pair displayed in the right column.

- **Question:** The question text shown in the accordion header.
- **Answer:** The answer content revealed when expanded. Supports rich text formatting.
