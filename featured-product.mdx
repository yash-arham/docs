title: "Featured product"
---

The **Featured Product** section lets you spotlight a single product in a full editorial layout directly on any page. It renders the product's media gallery alongside the complete buy box — including variant selectors, pricing, and add-to-cart — giving customers everything they need to purchase without navigating to the product page.

</Frame>

### How to Add the Section

1. From your Shopify admin, go to **Online Store → Themes** and click **Customize**.
2. Navigate to the page where you want to add this section.
3. Click **Add section** and select **Featured product**.
4. Select a product and configure the display settings.
5. Click **Save**.

### Section Settings

- **Product:** Select the product you want to feature in this section.
- **Color scheme:** Choose a color scheme to control the section's background and text colors.
- **Enable sticky info:** When enabled, the product information column stays fixed on screen as the customer scrolls through the media gallery, keeping the buy box always in view.

### Block Settings

This section uses the theme block system to build the buy box. You can add and reorder the following blocks within the section:

- **Product media:** Renders the product's full media gallery — images, videos, and 3D models.
- **Custom liquid:** Insert custom Liquid code anywhere in the buy box for advanced use cases.
- **@theme blocks:** Add any compatible theme blocks to extend the buy box layout.
- **@app blocks:** Add blocks from installed Shopify apps (e.g., reviews, loyalty programs).
