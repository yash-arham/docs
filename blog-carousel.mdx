The **Blog Carousel** section displays a horizontally scrollable row of article cards from a selected blog. Each card shows the article's image, category, title, and a read link. It's ideal for surfacing recent journal posts on the homepage, campaign landing pages, or between product sections.

</Frame>

### How to Add the Section

1. From your Shopify admin, go to **Online Store → Themes** and click **Customize**.
2. Navigate to the page where you want to add this section.
3. Click **Add section** and select **Blog carousel**.
4. Select a blog, configure the display settings, and click **Save**.

### Section Settings

#### Content

- **Eyebrow:** Small label displayed above the heading.
- **Heading:** Main section heading.
- **Heading typography:** Select the typographic style for the heading.
- **Read label:** The text shown on the read link for each article card (default: "Read →").
- **View all URL / View all label:** Link and label for a "view all" CTA displayed at the end of the carousel.

#### Source

- **Blog:** Select the blog to pull articles from.
- **Posts to show:** Number of article cards to display (range: 3–10).

#### Display

- **Columns desktop:** Number of article cards visible at once on desktop (2, 3, or 4).
- **Image ratio:** Aspect ratio for article card images — Portrait, Square, Landscape, or Adapt.

#### Placeholder Content *(editor only)*

- **Placeholder category / title / excerpt:** Fallback text shown in the theme editor when no blog is selected.

#### Style

- **Color scheme:** Choose a color scheme for the section.

#### Spacing

- **Padding top / Padding bottom:** Control vertical spacing around the section.
