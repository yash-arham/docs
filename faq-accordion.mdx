title: "FAQ accordion"
---

The **FAQ Accordion** section is a categorized accordion-style FAQ layout with a contact details sidebar and CTA. It supports category separator headings to group related questions, making it well-suited for dedicated FAQ pages with multiple topic areas and a clear path to direct customer support.

</Frame>

### How to Add the Section

1. From your Shopify admin, go to **Online Store → Themes** and click **Customize**.
2. Navigate to your FAQ page.
3. Click **Add section** and select **FAQ accordion**.
4. Add **Category** blocks as topic separators, then add **Question** blocks beneath each category.
5. Click **Save**.

### Section Settings

#### Content

- **Eyebrow:** Small label displayed above the heading.
- **Heading:** Main section heading.
- **Heading typography:** Select the typographic style for the heading.
- **Body:** Supporting text displayed below the heading.

#### Sidebar

- **Contact 1–4 (Label / Value):** Up to four contact detail rows displayed in the sidebar (e.g., "Email / hello@brand.com", "Phone / +1 212 000 0000").
- **CTA text / URL:** A call-to-action button label and link destination displayed below the contact details.

#### Behaviour

- **First open:** When enabled, the first accordion item starts in the expanded/open state.

#### Style

- **Color scheme:** Choose a color scheme for the section.

#### Spacing

- **Padding top / Padding bottom:** Control vertical spacing around the section.

---

### Category Block Settings

A Category block acts as a visual topic separator between groups of questions.

- **Name:** The category heading text displayed above its group of questions.

---

### Question Block Settings

Each Question block is one accordion question-and-answer pair.

- **Question:** The question text displayed in the accordion header.
- **Answer:** The answer revealed when the accordion is expanded. Supports rich text formatting.
- **Link 1 text / URL:** An optional first related link shown within the answer.
- **Link 2 text / URL:** An optional second related link shown within the answer.
